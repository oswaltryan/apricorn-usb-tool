# Windows Installer

This directory contains the files needed to build a `.exe` installer for Windows.

## Prerequisites

- Windows
- [NSIS](https://nsis.sourceforge.io/Download)

## Build Instructions

1.  Build the `usb.exe` executable by running `build\\build_windows.bat` from the root of the project.
2.  Right-click on `installers\\windows\\usb-tool.nsi` and select "Compile NSIS Script".
